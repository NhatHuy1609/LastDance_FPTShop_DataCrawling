## Installation

### Windows

```bash
# Create virtual environment
python -m venv env

# Activate virtual environment
env\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

### Linux

```bash
# Create virtual environment
python -m venv env

# Activate virtual environment
source env/bin/activate

# Install dependencies
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
